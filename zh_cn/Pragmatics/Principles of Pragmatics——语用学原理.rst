Principles of Pragmatics
========================

By Geoffrey. N. Leech, Longman, 1983, Pp. xii, 250, ISBN O 582 55110 2.

本书作者到莱是我们比较熟悉的一位英国语言学家,他于1980年出版了《语义学与语用学之探索》(Explorations in Semantics and Pragmatics) 一书,1981年发表了《语用学与交际中的修辞》(Pragmatics and Conversational Rhetoric) 一文,在此基础上于 1983 年出版了《语用学原理》(Principles of Pragmatics) 一书,进一步阐述了对语用学的看法。本书主要特点是通过对语言学两大学派——乔姆斯基转换生成理论和韩礼德 (M.A.K. Halliday) 系统语法理论作了分析、肯定后,结合两个学派的特点,以奥斯订一合尔言语行为理论为基础,扩大发展了格赖斯 (H.P. Grice) 的'会话含义理论',提出了一个建立在语言功能基础上语用学模式。

本书除序言外,共分十章。第一章 Introduction, 第二章 A set of postulates, 第三章 Formalism and functionalism, 第四章 The interpersonal role of Cooperative Principle, 第五章 The Tact Maxim, 第六章 A survey of Interpersonal Rhetoric, 第七章 Communicative Grammar: an example, 第八章 Performatives, 第九章 Speech-act verbs in English, 第十章 Retrospect and prospect.

在本书序言中,作者给语用学下了如下定义: 语用学是研究语语在环境中如何具有某意思的。作者认为语言的用法与作为抽象体系的语言本身有区别,但两者又互为补充,具体地说,语法必须与语用学区别开来。

第一章简要地回顾了语用学发展的历史,提出了八条准则,作为本书讨论的中心。准则1:句子的语义表达与语用装运有区别;准则2:语义学由语法框则牵强,普通语用学由修辞原则控制;准则3:语法准则主要是传统的,普通语用学原则主要是非传统的,即以达到交际目的为动力的;准则4:普通语用学结语语的语法含义 (sense) 与其语用意义 (force) 结合起来。这两者之间的关系可能是相对中立或或相对间接的;准则5:语法的一致性是由相互连系统决定的,语用的一致性由问题以及解决问题的方法决定的;准则6:语法解释基本上是正式的,语用解释是功能性的;准则7:语法依附于概念,语用学依附于人与人之间的关系以及上下文之间的类系;准则8:一般来说,可以对语法进行抽象、具体的描述和分类,对语用学可以描述其不同的、非具体的涵义。作者接着用动词 to mean 来举例说明语义学与语用学之间的不同之处。①X是什么意思?(What does X mean?) ②你用X来示什么意思? (What did you mean by X?)

语义学是以二元论的观点处理意思的,如句子①,而语用学则以三元论的观点解释意思的,如句子②。考虑到社会语用学在不同的文化、语言以及社会环境中解释不同,作者因而提出下面一个比较抽象的普通语用学模式,并认为此模式对于语言作普遍性的研究是有必要的。

::

    普通语用学
        |
    ________________
    |                |
    [语法]   语用语音学   社会语用学    [社会学]
    人相互连系            相互连系

第二章就语法准则与语用原则、语法含义与语用涵义之间的区别进行了讨论。对语法准则可以分成类别进行抽象的描述,而语用原则却不能分成类别。语用涵义是超过句子语法含义之外的"实际意思",例如 Can you pass me the salt? 的语法含义只是说明这是一个问句形式(句法解释),询问听话者的某种能力(语义解释),而这句话的语用涵义是,这是一个"请求句",委婉地请求听话者去完成某件事。因而,在实际交际中,说话者与听话者双方各有各自的任务,如何最有效地传递某个信息,使这个信息被听话者接收理解(讲话者的任务),怎样准确地理解解释传递来的信息,而去做出相应的反应(听话者的任务)。

作者在第三章中以形式语法学的观点解释语法,以功能学说的观点分析语用学,并指出,以形式一功能的观点来解释语言是比较切实可行的。他认为,语言是由语法和语用学组成,作为抽象形式认为,由语法输出并解释信息,由策略与原则组成的体系即语用学来加工,最后由语法和语用学则通过语法的帮助保证交际顺利进行。

第四章主要就作者作为对格赖斯合作原则学说的增补所提出的'礼貌原则'(Politeness Principle)的详细的阐述。人们在交际中不仅仅遵循合作原则,同时也遵循礼貌原则,以达到更好的交际效则,例如主人邀请客人,在下面常用的四句话中,

- ③Will you have anything to eat?
- ④Will you have something to eat?
- ⑤Won't have anything to eat?
- ⑥Won't you have something to eat?

句子 ⑥是最客气的说法,因为句子 ⑥否定了听话者客气的这想(即听话者想要谢拒讲话者的邀请),同时说话者作出同样客气的想法(不接受听话者的讲),终句话说,在交际中,讲话者注意遵循礼貌原则,讲话者认为听话者也注意遵循礼貌原则,讲话者肯定听话者在认为讲话者注意遵循礼貌原则。这在语用学中是一个很重要的内容。

第五章对礼貌原则中的一个组成部分——策略原则进行了解释。策略原则是人们交际中常常遵循原则之一,如何以最经济、最省有表达力而又没各气的语言方式来传递信息。作者提出了一个"付代价一受益"衡量表,句子的含义对听话者是受益的话,这个句子就越客气,反之相反,如

| ①Peel these po-<br>tatoes. | Cost to H    | Less polite |
| -------------------------- | ------------ | ----------- |
| ②Hand me the<br>newspaper. | ↑            | ↑           |
| ③Sit down.                 |              |             |
| ④Look at that.             |              |             |
| ⑤Enjoy your<br>holiday.    |              |             |
| ⑥Have another<br>sandwich. | Benefit to H | More polite |

第六章进一步就礼貌原则及其它一些原则如'大方原则',缩小对自己的受益程度,扩大对自己的付代价程度;"满意原则",缩小对他人的非难,增加对他人的赞扬;"谦让原则",减少对自己的赞扬,增大对自己的自责;"呼应原则",增大双方的一致点,减少双方的分歧点,等等进行讨论。这些原则与礼貌原则等组成了人们在交际中所采用的修辞法。

第七章以交际语法为例子,进一步说明了将语义学与语用学区别开是合理的,因为句子意思在语义与语用两个不同方面都有各自要遵循的规则。在语义这一层上,可以将语法规则公式化,用来解释肯定与否定,陈述式与疑问式,陈述与问句等关系句子之间的逻辑关系,还可以解释这些类型句子在句法上的不对称现象。在语用学这一层上,合作原则与礼貌原则决定语言的表达形式能与交际功能联系起来。这两种原则所采用的方法可以归纳并形成一般性的结论。作者回时还指出,语用学没有必要将言语行为的规则包括进去,如合尔 (J. R. Searle) 所提出的言语行为规则说。

第八章陈述了奥斯订 (J. L. Austin) 与各尔关于言语行为学说的长处与短处。作者提出了与言语行为学说出发点不同的看法,对于动词意思的研究并不是作为了解语用涵义的'关键',而是作为帮稠人们是如何去谈论这些语用涵义的'关键'。

第九章主要讨论了言语行为动词。作者首先回顾了奥斯订对言语行为所做的分类,句子表达方式 (locution);句子语用涵义 (illocution);句子语用涵义所引起相应的结果 (perlocution)。作者认为除了这三种言语行为的分类外,还应加上一类——语言行为,因为语语的出发点是从语言行为开始的。作者又总结了语用行为动词通常在哪些句类中出现,例如陈述类动词通用于S(讲话者)+动词+内容/主题,常用动词有 affirm, allege, assert, predict, insist, announce 等;指示类动词常出现在 S (讲话者) +动词+ 宾语,常用动词有 ask, beg, bid, command, demand, forbid, recommend, request 等;表态类动词常用于 S (讲话者)+ 动词+ 从句,常用动词有 offer, promise, swear, vow, volunteer 等;表达类动词多用于 S (讲话者)+动词+介词/宾语,常见动词有 apologize, commiserate, congratulate, pardon, thank 等。然后作者分别就这四种类形动词举例加说明。

第十章简要地总结了前九章中所讨论的要点并指出,许多与这些要点有紧密联系的问题没有全部包括在本书中,作者将这些问题列在八点建议中,包括对这八点所包括的方面进行讨论,以更好地理解语言的本质,为交际服务。

《语用学原理》这理论上是比较深一点儿。但从语言教学的角度看,本书中对于语法与语用学之间的关系进行较系统的解剖、分析,以交际语法为例来说明两者的关系,因此对于从事外语教学的教师来说,是很有参考价值的。

(孙建荣)
