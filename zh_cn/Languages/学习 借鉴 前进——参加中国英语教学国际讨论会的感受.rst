
学习 借鉴 前进
==============
**参加"中国英语教学国际讨论会"的感受**

**杜瑞清 孙建荣**

经过几年的酝酿和筹备,中国英语教学国际讨论会(International Symposium on the Teaching of English in the Chinese Context)于1985年9月23日——27日在广州外国语学院召开。在郁郁葱葱、风景如画的羊城,来自英、美、澳等国家和港、澳地区的二十三位客人与国内五十多所大专院校的教授专家济济一堂,共同探讨了有关中国英语教学的现状、经验和问题。这是建国以来,首次举办的由中外学者共同参加的英语教学讨论会,它必将对我国英语教学的研究和发展起推动作用。

与会代表在"中国英语教学的现状及展望"这一总题目下,就专业英语教学的特点和问题,交际法在我国的应用,文化知识课、文学课在英语教学中的地位,科技英语教学的主要任务以及英语测试方法的改进等问题,广泛地交换了意见。

我们有幸参加了这次讨论会,受到了启发,学习了不少东西。下面是我们的几点感受。

**一、"传声筒"时代即将过去**

党的十一届三中全会以来,我国政治、经济形势发生了根本变化,我国的国际影响日益扩大,对外开放,对内搞活的政策,调动了各方面的积极性。四化建设需要一代新的外语工作者,他们不但要外语基本功过硬,而且知识面要广,政策性要强。现在已经出现了这样的势头,在不远的将来,外语工作者将把外语作为交际工具,进行直接采访、外事谈判和贸易谈判,获取科技信息,进行文化交流,仅仅当个翻译,只起"传声筒"的作用的时代即将过去。新的形势,对外语教学提出了更高的要求。时代在变,形势在变,学生在变,我们必须跟上形势的要求,坚决贯彻"面向四化,面向世界,面向未来"的方针,不断解决外语教学中的新问题,把外语教学提高到一个新的水平。

**二、吸收各家之长,创中国外语教学的新路**

近年来,语言学的研究成果已渗透到英语教学的各个方面,各种不同流派的教学法应运而生,这对英语教学的发展,起了重大推动和指导作用。我们应当,而且必须加强对教学法的研究,吸收各家之长,推动我国外语教学的发展。目前在外语教学中,交际法相当流行。然而对交际法的实质,并非人人都了解。人们往往把那种俏皮嘻嘻元味和明知故问的句型操练认为是交际法,或者只把交际法局限于一般的口语会话。其实,这是一种误解。交际法的采用、发展与社会语言学、话语分析和文体学等学科的发展密不可分。它一方面强调给学生教授真实的语言,另一方面强调以学生为中心。交际包括人与人之间一切思想感情的交流,是一种活的过程。从一定意义上说,在交际活动中,过程比结果更重要,这就要充分发挥学生的主观能动性,教师只是组织指导学生进行活动,而不能包打天下。交际法既避免了语法——翻译法的弊病,并克服了听说领先法的片面性,是听说读写一起抓,强调有意义的真实的交际。正如有些专家指出的:"交际教学法就其性质而言,本来就不是什么定型的教学法,而是潜力无限、变化无穷的一种教学路线。"广州外语学院经过几年的实践,已取得初步经验,值得我们学习。

教学法,归根结底,只是一种方法。世界上没有一种包医百病的良药,也没有一种完美无缺的教学方法。在教学方法的研究和采用方面,我们既不能因循守旧,也不能盲目接受。明智和有效的办法应该是博采众家之长,避免各派之短,不妨来个综合应用。这就要求我们了解各家,研究各家,在充分了解和研究的基础上,决定取舍,走出自己的路来。正象一位语言学家建议的那样,要 Adopt non-inclusively, reject non-exclusively.

**三、加强对各国文化的学习和研究**

语言是文化的有机组成部分。学习外国语势必要学习了解外国文化。有人把文化比做森林,把语言比做树林,学习外语既要见树木,更要见森林。日益频繁的对外交往,使人们愈来愈认识到了解西方文化传统和风俗习惯的重要性。只机械地学习词汇、语法、句型,即使是学得再好,也往往达不到预期的效果,甚至事与愿违,这类事例屡见不鲜。许多学者建议在教学中自觉地引导学生注意中西文化的异同,培养敏锐的观察能力、分析能力和接受能力,也有人指出 To be really bilingual you've got to be bicultural.

谈起文化,英语中有小写 c 和大写 C 的区别。这里的 c 代表 culture(文化)。小写 c 一般指从人类学的角度研究文化,即风俗习惯、社交特点、日常语言应用特征等,这些在已往的英语教学中常常被忽略。大写 C 主要是指社会历史、科学技术、文学艺术等等。英语专业的文化课、阅读课、文化知识课等包括了这一方面的内容。总而言之,语言和文化密不可分,要提高英语教学质量,必须把英语国家的文化的研究提高到应有的高度来认识。

**四、新时期对师资队伍建设的新要求**

许多与会代表在发言中强调了培养教师的重要性。任何科学的教学方法,先进的教材和现代化的教学设备都是靠教师来使用的。从宏观的角度看,一个合格的外语教师,应该具备扎实的专业基础、良好的素质训练和较强的科研能力。当前的状况是很多同志专业基础不错,但在后两方面比较薄弱。经验证明,学术造诣和合格教师是既有联系又有区别的两码事,既不能划等号,也不能相互代替。因此,对教师来说,应尽可能学习、掌握教育学、教育心理学、教学法,增强自己的科研能力。这是从总的方面提出问题,有的语言学家进一步从微观角度提出对教师的十一条要求:一、具备英语的基础知识。二、具有使用英语进行教学工作的能力。三、具备用理论知识来指导实际教学与学识。三、具有使用英语进行教学工作的能力。三、具备用理论知识来指导实际教学与学习的能力。四、具备能以口头和笔头形式向不同的教学对象传递信息的技巧。五、能够评价不同种类的教材,以便进行合理的吸收、修改,使之更好地适用于具体的教学环境和教学对象。六、具有分析教学对象对教学的需求以及他们自身的长处和短处等的能力。七、在教学中充分使用现有各种教学设备的能力。八、具备使用除教材以外的其它参考材料而需要的知识。九、根据教学对象在课堂上组织适合于不同水平的教学活动。十、具备自己设计、安排每一课的计划、顺序、课程的组成甚至整个一个课程大纲的设制等方面的能力。十一、具备自我检查教学效果的能力。

形势在变化,时代在发展。我们也必须善于学习,既了解国外的最新动态和成果,也学习和借鉴国内兄弟院校的经验,结合实际,采取切实可行的措施,把我院的教学和科研工作推向前进。
