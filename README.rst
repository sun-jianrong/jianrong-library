Getting Started
================

The Getting Started Guide provides step-by-step instructions for installing, configuring, and building the Beanable SDK for macOS, Linux, and Windows. It also provides instructions for keeping the Beanable SDK tools and build environment up to date over time.

Guides for working with specific microcontrollers supported by the Beanable SDK are in the devices directory. These include links to additional detail for specific development boards including Beanable S32, Beanable S32-S3, etc.

入门指南
================

入门指南提供了安装、配置和构建macOS、Linux和Windows的Beanable SDK的分步说明。

它还提供了如何使Beanable SDK工具和构建环境随时保持最新的说明。使用Beanable SDK支持的特定微控制器的指南在devices目录中。这些链接包括特定开发板的其他详细信息，包括Beanable S32, Beanable S32- s3等。
